You can replace the file by changing line 10 in the code:```
with open("text.txt", "r", encoding="utf-8") as f``` just put the name of the file you want to use (```only .txt```).**

### how to download?
**Download all components, including the requirements file.
Then open the console (```Win+R``` → ```cmd```), navigate to the directory where the requirements file is located, and run the following command:
```pip install -r requirements.txt```

```python troll_hidlow.py``` or ```py troll_hidlow.py```**
